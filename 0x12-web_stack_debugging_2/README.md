Project Overview
This project focuses on debugging web stack issues, particularly ensuring that processes run with the appropriate user permissions to enhance security on Unix systems.

Files and Their Functions
0-iamsomeoneelse
Function: This Bash script accepts one argument and runs the whoami command under the user passed as an argument.
1-run_nginx_as_nginx
Function: This Bash script configures the container to ensure that Nginx runs as the nginx user and listens on port 8080 on all active IPs.
100-fix_in_7_lines_or_less
Function: This Bash script provides a concise solution (7 lines or less) to ensure Nginx runs as the nginx user, improving the solution from task #1.
